Create repository on github account.

On the IDE terminal,

# Initialize new Git repository
git init

# Set global Git username
git config --global user.name "Tao"  # This is the username of your github account

# Set global Git email
git config --global user.email "aribs@gmail.com"  # This is the email of your github account

# Check current status of files
git status          

# Stage file for commit
git add ziko_logistics_data.csv

# Commit staged changes with message
git commit -m "first commit"

# Add remote repository named 'origin' for pushing/pulling changes
git remote add origin https://github.com/Taoficoda/ziko_logistics

# Push local 'master' branch to 'origin' remote and set upstream tracking
git push -u origin master    --- Note 'master' is the name of the local or source branch we are working on.

# Add all files to repo, the dot(.) means all files
git add .
